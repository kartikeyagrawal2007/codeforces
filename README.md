# Codeforces Solutions

This repository contains my solutions to various [Codeforces](https://codeforces.com/) problems, categorized by their difficulty ratings.

## 📂 Structure

The repository is organized into folders based on problem difficulty ratings:
- `800/` - Solutions for problems rated 800
- `900/` - Solutions for problems rated 900
- `1000/` - Solutions for problems rated 1000
- ...and so on.

## 🛠️ Language

- **C++**: The solutions are primarily implemented in C++.

## 🚀 How to Run

To compile and run a C++ solution locally, you can use `g++`:

```bash
# Compile the solution
g++ -o solution problem_name.cpp

# Run the executable
./solution
```

## 📝 Note

These are my personal solutions for learning and practice. There might be multiple ways to solve a problem, and some solutions may not be the most optimal. Happy coding!
